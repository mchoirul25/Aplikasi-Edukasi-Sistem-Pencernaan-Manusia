Aplikasi edukasi interaktif berbasis Android yang dikembangkan menggunakan Unity Engine (C#). Fokus utama proyek adalah gamifikasi pembelajaran sistem pencernaan manusia melalui kuis 3 level berjenjang dan sistem ranking kompetitif. Mengimplementasikan Algoritma Fisher-Yates untuk pengacakan soal yang adil, memastikan kualitas dan efektivitas aplikasi yang telah divalidasi melalui studi kasus lapangan.

    - Algoritma Pengacakan Soal: Implementasi Algoritma Fisher-Yates dalam logika kuis untuk memastikan urutan soal yang acak dan unik di setiap sesi.

    - Panduan Penggunaan: Fitur petunjuk komprehensif untuk memudahkan navigasi di dalam aplikasi.
